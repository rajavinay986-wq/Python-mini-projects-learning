salary-data-analysis 

This is a beginner-friendly Python project to analyze salary trends .

## What this project does
- Loads salary data from a CSV file
- Uses NumPy for basic statistics
- Uses Pandas for grouping and analysis
- Uses Matplotlib (and optional Seaborn) for visualizations

## Insights
- Average salary by city
- Salary distribution
- Relationship between experience and salary

## Tech Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
