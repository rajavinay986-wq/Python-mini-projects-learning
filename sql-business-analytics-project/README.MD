# SQL Business Analytics Project

## Overview
This project demonstrates real-world SQL usage for business analytics using SQL Server Management Studio (SSMS).

The project simulates an e-commerce business database and answers practical business questions using SQL queries.

---

## Tools Used
- SQL Server Management Studio (SSMS)
- SQL Server
- GitHub

---

## Database Schema
The database contains the following tables:

- Customers
- Products
- Orders
- Order_Items
- Payments

---

## Files in This Repository

- `schema.sql`  
  Contains all CREATE TABLE statements.

- `insert_data.sql`  
  Contains INSERT statements to populate the database.

- `analysis_queries.sql`  
  Contains business analytics SQL queries.

- `screenshots/`  
  Contains screenshots of query outputs from SSMS.

---

## Business Questions Answered

- Who are the top customers by total spending?
- What is the monthly revenue trend?
- Which products are the best sellers?
- Revenue distribution by country
- Customer ranking using window functions
- Analysis of failed and pending payments

---

## Skills Demonstrated

- SQL Joins (INNER, LEFT)
- GROUP BY & HAVING
- Subqueries
- Window Functions (RANK)
- Business-focused data analysis
